## 1. 이론적 배경: 체적·표면적과 생물자원 가공의 관계

### 1-1. 체적과 표면적의 가공학적 중요성
- **표면적**: 건조·냉각 공정에서 열 및 물질 전달 방정식의 경계 조건
- **체적**: 저장 시설 용량 설계 및 밀도(Density) 계산의 핵심 인자
- **비표면적(Specific Surface)**: 동일 건조 조건에서 비표면적이 클수록 건조 속도가 빠름
  - 예: 밀(1,316 m⁻¹) > 옥수수(728 m⁻¹) > 콩(558 m⁻¹)

### 1-2. 2주차 학습 내용과의 연계
- **2주차**: 원형도(Circularity), 구형도(Sphericity) — 형상의 **정적 지표**
- **3주차**: 체적, 표면적 — 형상의 **동적 물리량**으로 확장
- 구형도가 높은 사과 → 단순 공식으로도 체적 추정 정확도 높음
- 구형도가 낮은 곡립 → 더 정교한 수치 적분 모델 필요

### 1-3. 물리적 계측법과 그 한계
| 계측 방법 | 적용 대상 | 원리 | 한계 |
| --- | --- | --- | --- |
| **액체 치환법** (물) | 감자, 아보카도 등 | 아르키메데스 원리 | 수분 흡수가 빠른 시료 부적합 |
| **공기 피크노미터** | 곡물 (쌀, 밀 등) | 보일의 법칙 | 공기 압축성·마찰 오차 |
| **껍질 면적 측정** | 과실류 | 껍질 박리 후 면적계 | 파괴적 측정, 비재현성 |

### [요약: 세그먼트 모델링]
불규칙한 형상의 생물체를 반구, 원뿔대, 구의 일부분 등의 기하학적 요소로 분할하여 합산 — 파이썬 수치 적분으로 자동화 가능

## 2. 수치 적분 알고리즘의 이론적 기초

### 2-1. 회전체의 체적 (Volume of Revolution)
외곽 함수 `r(x)`를 중심축에 대해 회전시킨 체적:
```
V = π ∫₀ᴸ r(x)² dx
```

### 2-2. 회전체의 표면적
```
S = 2π ∫₀ᴸ r(x) × √(1 + (dr/dx)²) dx
```

### 2-3. 수치 적분 비교
| 방법 | 근사 방식 | 정확도 | SciPy 함수 |
| --- | --- | --- | --- |
| **사다리꼴 공식** (Trapezoidal) | 직선(1차) 근사 | O(h²) | `scipy.integrate.trapezoid` |
| **심슨 공식** (Simpson's) | 포물선(2차) 근사 | O(h⁴) | `scipy.integrate.simpson` |

### 3-1. [`step1_interpolation.py`](step1_interpolation.py): 프로파일 데이터 입력 및 큐빅 스플라인 보간
교재 예제 3-3의 아보카도 길이 방향 반지름 측정값을 입력하고, 큐빅 스플라인(Cubic Spline) 보간법으로 매끄러운 외곽 곡선을 생성합니다.

```python
import numpy as np
from scipy.interpolate import CubicSpline
import matplotlib.pyplot as plt

# 아보카도 예제 데이터 (위치 x [cm], 반지름 r [cm])
x_points = np.array([0, 1.0, 5.0, 6.5, 8.0, 10.85])
r_points = np.array([0, 1.47, 3.185, 3.38, 3.04, 0])

# 큐빅 스플라인 보간으로 매끄러운 곡선 생성
cs = CubicSpline(x_points, r_points, bc_type='natural')
x_new = np.linspace(0, 10.85, 100)
r_new = cs(x_new)
r_new = np.maximum(r_new, 0)  # 음수 반지름 보정

# 시각화: 측정점 vs 보간 곡선 (상하 대칭 단면)
plt.fill_between(x_new, r_new, -r_new, alpha=0.3, color='green')
plt.plot(x_points, r_points, 'ro', markersize=8, label='측정 데이터')
plt.title('아보카도 프로파일 - 큐빅 스플라인 보간')
plt.xlabel('길이 x [cm]'); plt.ylabel('반지름 r [cm]')
plt.legend(); plt.grid(True, alpha=0.3); plt.axis('equal')
plt.show()
```

**[학습 포인트]**
- 6개의 이산 측정점이 100개의 연속 포인트로 변환되는 과정 관찰
- 선형 보간(직선 연결)과 큐빅 스플라인(곡선 연결)의 차이를 시각적으로 비교

### 3-2. [`step2_volume.py`](step2_volume.py): 수치 적분을 이용한 체적 추정
보간된 외곽 함수 `r(x)`를 이용하여 각 위치에서의 단면적 `A(x) = π × r(x)²`을 계산하고, Simpson 공식과 사다리꼴 공식으로 체적을 산출합니다.

```python
from scipy.integrate import simpson, trapezoid

# 단면적 계산 (A = π × r²)
areas = np.pi * (r_new ** 2)

# 심슨 공식을 이용한 수치 적분
vol_simpson = simpson(areas, x=x_new)
print(f"추정 체적 (Simpson's Rule): {vol_simpson:.4f} cm³")

# 사다리꼴 공식을 이용한 수치 적분
vol_trapezoid = trapezoid(areas, x=x_new)
print(f"추정 체적 (Trapezoidal Rule): {vol_trapezoid:.4f} cm³")
```

**[학습 포인트]**
- 분할 수(n)를 10 → 20 → 50 → 100 → 1000으로 변화시키며 오차 수렴 관찰
- 동일 분할 수에서 Simpson이 Trapezoidal보다 높은 정확도를 보이는 이유 고찰
- 교재 수동 계산(5개 구간 분할)과 파이썬 적분(100개 이상 구간)의 정밀도 차이 비교

### 3-3. [`step3_3d_visualization.py`](step3_3d_visualization.py): 3D 회전체 표면 재구성 및 시각화
2D 프로파일 데이터가 3D 공간의 물체로 변환되는 과정을 Matplotlib의 mplot3d 툴킷으로 직관적으로 확인합니다.

```python
from mpl_toolkits.mplot3d import Axes3D

theta = np.linspace(0, 2 * np.pi, 50)
X, THETA = np.meshgrid(x_new, theta)
Y = cs(X) * np.cos(THETA)
Z = cs(X) * np.sin(THETA)

fig = plt.figure(figsize=(10, 8))
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z, cmap='YlGn', alpha=0.8)
ax.set_title(f"Reconstructed 3D Shape of Avocado (V={volume:.1f} cm³)")
plt.show()
```

**[학습 포인트]**
- 자신이 구현한 알고리즘이 실제 아보카도의 형상을 얼마나 잘 모사하는지 관찰
- 표면의 불규칙성이 적분 결과에 미치는 영향 토론
- 표면적 계산 결과(`S = 2π ∫ r(x) × √(1 + (dr/dx)²) dx`)도 함께 확인

### [실습 고찰 포인트]

#### A. 수치 적분의 수렴성
- 분할 수 n이 증가할수록 Simpson과 Trapezoidal의 차이가 0에 수렴
- n=100 이상이면 대부분의 농산물 형상에서 충분한 정밀도 확보 가능

#### B. 보간법의 선택이 결과에 미치는 영향
- 선형 보간: 계산이 간단하나 곡면 표현이 부자연스러움
- 큐빅 스플라인: 매끄러운 곡선으로 실제 생물체 형상에 더 근접

#### C. 4주차 밀도 학습과의 연결
- 본 실습의 체적 데이터는 4주차 진밀도(True Density) 및 산물밀도(Bulk Density) 계산의 기초
- `밀도 = 질량 / 체적` — 정확한 체적 추정이 밀도 정확도를 결정
